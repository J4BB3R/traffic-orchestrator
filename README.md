# Traffic Orchestrator
A simple mathematical implementation for road traffic orchestration simulation who can manage a lot of cars.

### HOW-TO :
- Install ```Docker``` and ```Docker Compose```
- Launch in shell ```docker-compose up```
- Inject your data with an app that's compliant with my REST API. RTFM ;)
- Open ```http://$_YOURHOST:8080``` and push the 'Start Track' button
